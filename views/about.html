About Hacking Corp

The Hacking Corp. web application was developed by the Hacking Pirates using the latest technologies in automation, continuous integration, cloud computing, testing and Single Sign-on (SSO). The Hacking Corp app provides user with information about the company and provides employees with access to the corporate database where they can see Products, Orders, Customers and Employees information. The Hacking Corp is example of a migration from traditional IT infrastructure to a cloud base infrastructure with modern development technologies.
